# Data Dictionary

This project focuses on the following key tables from the Olist dataset:

| Table Name | Description |
| :--- | :--- |
| `olist_orders_dataset` | Core table containing order status and timestamps (purchase, delivery, etc.). |
| `olist_customers_dataset` | Customer unique IDs and location (city/state). |
| `olist_order_items_dataset` | Details of products within each order, including price and shipping cost. |
| `olist_order_reviews_dataset` | Customer satisfaction scores (review_score). |
| `olist_geolocation_dataset` | Coordinates and zip codes for spatial mapping. |
| `product_category_name_translation` | Translates Portuguese category names to English. |
