# Dataset Information

The raw data used in this project is sourced from the **Brazilian E-Commerce Public Dataset by Olist** available on Kaggle.

Due to GitHub's file size limitations, the raw CSV files are not uploaded to this repository. You can download the original dataset directly from the link below:

[Kaggle: Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

### Instructions for Local Setup:
1. Download the dataset from Kaggle.
2. Extract the files into your local `/data/` folder.
3. Ensure all CSV filenames remain unchanged for the Power Query connection.
